Predict the Flat Resale Prices in Singapore¶

1 Problem Description¶

Investing in Flats is the biggest investment for most households in cities like Singapore which is very small but heavily urbanised. Therefore, being able to accurately value the flat prices will not only facilitate market transactions by providing valuable guidance for all market participants (be it home owners, home buyers, landlords, tenants or banks that underwrite mortgages), but also provide useful insights for policy makers and government authorities in understanding the current state of the economy.

In this competition, you are expected to create an analytical and modelling framework to predict the flat resale prices based on the quantitative and qualitative features provided in the dataset while answering other questions too cited below.

Evaluation Metric: Consider the ‘RMSE’ as the error metric for regression task to tune the model¶
For Information: Singapore Flats have 99-year lease¶

3 Thought Process¶

  • remaining_lease : The data for reamining lease in:
    • train1 : remaining_lease column is missing in this dataset so it'll be calculated based on the information given to us.
    • train3 and test: remaining_lease have have value in months and year so the month part will be removed so that we can match the data as we have it in test2.
  • Further Data can be pre processed after we merge it together so we don't have to preprocess other files seperately.
  • Unique ID : Since it contains unique ID we can delete this column.
  • month : It has year-month values and can be seperated into two columns year and month.
  • town : Town also has a major impact on the resale price.
    • A 2 Room Flat will cost different in one city than the other.
    • We can calculate the median of the resale price based on the city. (We can also use mean/average or Average Price/Sq. Foot etc instead of mean.
  • storey_range : It has data in bins which can be converted into continuous numbers.
